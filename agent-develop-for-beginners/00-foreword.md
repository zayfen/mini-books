# 前言：为什么我们需要 AI Agent？

当我们第一次使用大语言模型（如 ChatGPT、Claude）时，往往会被它惊艳到了：它能写诗、写代码、甚至帮你构思小说。但用久了之后，你会发现它的**局限性**。

比如，你对它说：
> "帮我把这篇关于 AI Agent 的英文论文翻译成中文，然后按照指定格式发邮件给我的同事。"

由于它只是一个"在输入框里和你聊天的模型"，它无法真正打开你的邮箱，无法读取你本地的 PDF，也无法自己去处理格式排版。它只能给你一堆翻译好的文本，剩下的事情还得你自己一步一步动手。

**这就好比你雇佣了一个学富五车、极其聪明的顾问，但他被五花大绑在椅子上——除了能说话，什么事情都必须由你代劳。**

这就是为什么我们需要 **AI Agent（人工智能体）**。

## 🤖 什么是 AI Agent？

AI Agent 是以大语言模型（LLM）为“大脑”，并且装备了“手”（工具调用）、“记忆”（上下文管理）、“眼睛”（环境感知）的智能体。

有了 Agent 技术，之前的那个指令就会变成现实：
1. **感知**：Agent 自动读取你发送的 URL 或 PDF 文件。
2. **推理规划**：大脑（LLM）分析出需要分三步：阅读理解 -> 翻译 -> 发邮箱。
3. **行动**：调用代码或 API，自动完成邮件的撰写与发送。

如果说过去的一年是“大模型元年”，那么现在就是“智能体大爆发的时代”。
