# 第九章：实向量空间上的算子——实数世界的特殊情况

## 9.1 复化

研究实算子不好办，一个技巧是把它变成复算子：

```
复化 Vc = V + iV
```

### 🔄 小白理解

把实数世界的问题，先放到复数世界去处理，处理完再拿回来。

### 🎯 类比

就像处理困难问题时，先换个角度（复数视角），解决后再回到原来的角度。

---

## 9.2 重要结论

### 结论 1

> 每个实算子都有 **1 维或 2 维不变子空间**

### 结论 2

> 奇数维实空间上的算子**一定有本征值**

比如 3 维空间里的旋转一定有一条**不动轴**。

---

## 9.3 实正规算子的刻画

在实内积空间上，正规算子的矩阵可以写成**分块对角**，每块要么是：

### 类型 1：1×1（实数本征值）

```
[λ]
```

### 类型 2：2×2（旋转块）

```
⎡ a  -b ⎤
⎣ b   a ⎦   （b > 0）
```

### 💡 大白话

实正规算子就是一些**拉伸**和一些**旋转**的混合。

---

## 9.4 实等距同构的刻画

实等距同构的矩阵可以写成**分块对角**，每块是：

### 类型 1：恒等

```
[1]
```

### 类型 2：反射

```
[-1]
```

### 类型 3：旋转

```
⎡ cos θ   -sin θ ⎤
⎣ sin θ    cos θ ⎦
```

### 💡 大白话

实数世界的保长变换，就是**旋转**、**反射**和**恒等**的组合。

---

| 概念 | 通俗理解 |

| 实正规算子 | 拉伸 + 旋转 |
| 实等距同构 | 旋转 + 反射 + 恒等 |
