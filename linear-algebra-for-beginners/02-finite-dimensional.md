# 第二章：有限维向量空间——空间的大小

## 2.1 张成空间——用几个点画出一个平面

给你几个向量，你能用它们的各种组合得到哪些新向量？

所有这些能得到的向量的集合，就叫这组向量的**张成空间**。

### 🎨 直观理解

| 向量数量 | 能张成什么？ |
|----------|--------------|
| 1 个向量 | 一条直线 |
| 2 个不共线的向量 | 一个平面 |
| 3 个不共面的向量 | 一个立体空间 |

### 📐 例子

```
向量：(1, 0) 和 (0, 1)

张成空间 = 整个二维平面
因为任何点 (x, y) = x(1,0) + y(0,1)
```

---

## 2.2 线性无关——谁是多出来的？

一组向量如果其中**任何一个都不能用其他向量的组合表示出来**，它们就是**线性无关**的。

### 🔍 判断方法

| 情况 | 线性无关？ |
|------|------------|
| 两个向量不共线 | ✅ 是 |
| 两个向量共线 | ❌ 否（一个能用另一个表示） |
| 三个向量不共面 | ✅ 是 |
| 三个向量共面 | ❌ 否 |

### 🏠 生活类比

你要盖房子，需要三根不同方向的柱子。

如果第三根柱子能用前两根拼出来，它就是**多余的**，根本不需要。

---

## 2.3 基——最精简的生成器

**基**就是一组既能张成整个空间、又线性无关的向量。

### 📖 生活类比

基就像**字母表**：
- 26 个字母能拼出所有单词
- 而且每个字母都是必要的（不能少）

### 🎯 标准例子

二维空间的标准基是 `(1, 0)` 和 `(0, 1)`。

你可以用它们组合出任何点：

```
(x, y) = x(1, 0) + y(0, 1)
```

### ✅ 基的重要性

| 性质 | 说明 |
|------|------|
| 张成 | 能表示空间里任何向量 |
| 最小 | 少一个都不行 |
| 唯一表示 | 每个向量只有一种表示方式 |

---

## 2.4 维数——空间的大小

**维数**就是基中向量的个数。

### 📏 各种空间的维数

| 空间 | 维数 |
|------|------|
| 一个点 | 0 |
| 直线 | 1 |
| 平面 | 2 |
| 我们生活的物理空间 | 3 |

### 🔑 重要结论

> **任何有限维空间的所有基都有相同数量的向量**，所以维数是唯一确定的。

这意味着：不管你选哪组基，基的大小都一样！

---

## 📝 本章小结

| 概念 | 通俗理解 |
|------|----------|
| 张成空间 | 用几个向量能"画"出的所有东西 |
| 线性无关 | 没有任何一个向量是多余的 |
| 基 | 最精简的"生成器组" |
| 维数 | 基的大小 = 空间的"大小" |
