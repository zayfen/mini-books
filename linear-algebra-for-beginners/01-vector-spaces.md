# 第一章：向量空间——什么是"空间"？

## 1.1 从数字到"包裹"

想象你有一个快递单号，上面写着：`(2024, 北京, 张三)`。

这就是一个**有序组**——几个数字或信息按顺序排在一起。

- 两个数字的组 `(x, y)` 就是二维空间里的一个点
- 三个数字的组 `(x, y, z)` 就是三维空间里的一个点
- n 个数字的组就是 n 维空间里的一个点

我们把这种组叫做**向量**。

### 🔢 向量的例子

| 维度 | 例子 | 含义 |
|------|------|------|
| 2D | (3, 4) | 平面上的一个点 |
| 3D | (1, 2, 5) | 空间中的一个点 |
| 5D | (90, 85, 78, 92, 88) | 五门课的成绩 |

---

## 1.2 向量空间——一个大操场

向量空间就是一个集合，里面装满了这样的"包裹"（向量），并且你可以对它们做两件事：

### 操作 1：加法

把两个包裹加起来，得到一个新包裹：

```
(1, 2) + (3, 4) = (4, 6)
```

### 操作 2：数乘

把一个包裹放大或缩小：

```
3 × (1, 2) = (3, 6)
```

### ✅ 向量空间的定义

只要加法和数乘的结果还在这个集合里，这个集合就是一个**向量空间**。

> **最简单的例子**：整个平面上的所有点，就是一个二维向量空间。

---

## 1.3 子空间——操场里的小圈子

子空间是向量空间的一部分，但它自己也是一个合法的"小操场"。

### 必须满足三个条件：

1. **包含原点** `(0, 0)`
2. **加法封闭**：里面任意两个点相加，还在里面
3. **数乘封闭**：里面任意一个点放大缩小，还在里面

### 🎯 例子

| 是否是子空间 | 原因 |
|--------------|------|
| ✅ 过原点的直线 | 满足三个条件 |
| ❌ 不过原点的直线 | 不包含原点 |
| ✅ 过原点的平面 | 满足三个条件 |
| ✅ 只有原点 `{(0,0)}` | 最小的子空间 |

---

## 1.4 子空间的和与直和

### 子空间的和

把两个小圈子里的人全部拉出来，让他们自由组合（相加），得到的集合就是两个子空间的**和**。

```
U + W = {u + w | u 在 U 里, w 在 W 里}
```

### 直和（Direct Sum）

如果两个小圈子**除了 0 以外没有任何重叠**，它们的和就是**直和**。

这时候每个人都可以**唯一地**表示成两个圈子里的人之和。

> **生活类比**：两个部门的人合作，如果两个部门没有重叠人员，每个人都知道自己来自哪个部门。

---

## 📝 本章小结

| 概念 | 通俗理解 |
|------|----------|
| 向量 | 有序的数字组 |
| 向量空间 | 可以做加法和数乘的"大操场" |
| 子空间 | 大操场里合法的"小圈子" |
| 直和 | 两个不重叠的小圈子完美合作 |
