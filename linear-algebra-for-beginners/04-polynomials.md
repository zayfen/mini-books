# 第四章：多项式——数学里的"积木"

> 这一章不讲线性代数，只讲多项式。为什么要学？因为后面研究算子时，多项式是重要的工具。

## 4.1 复系数多项式的分解

每个复系数多项式都能写成：

```
p(z) = c(z - λ₁)(z - λ₂)...(z - λₘ)
```

其中 λ₁, ..., λₘ 是多项式的**根**。

### 🧱 生活类比

这就像把一个数**分解成质因数**一样。

### 📐 例子

```
p(z) = z² - 5z + 6
     = (z - 2)(z - 3)

根是 2 和 3
```

---

## 4.2 实系数多项式的分解

实系数多项式就不一样了——它的非实数根总是**成对出现**。

比如 `2+i` 和 `2-i` 一起出现。

### 📝 分解形式

所以分解式里会包含一些**二次因子**：

```
x² + bx + c （其中 b² < 4c）
```

### 🎯 例子

```
p(x) = x² + 1
     = (x - i)(x + i)

在实数范围内无法分解
```

---

## 📝 本章小结

| 概念 | 说明 |
|------|------|
| 复多项式分解 | 完全分解成一次因子 |
| 实多项式分解 | 有二次因子 |
| 根 | 让多项式等于 0 的值 |
