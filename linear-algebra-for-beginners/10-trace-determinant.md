# 第十章：迹与行列式——最后的拼图

> 终于到了最后！我们终于可以讨论行列式了。

## 10.1 迹

### 📐 定义

**迹** = 按重数计算的所有本征值之和 = 矩阵对角线元素之和

```
trace(T) = λ₁ + λ₂ + ... + λₙ
         = T₁₁ + T₂₂ + ... + Tₙₙ
```

### 💡 小白理解

就像给算子算个"**总账**"，把它的本征值加起来。

---

## 10.2 行列式

### 📐 定义

**行列式** = 按重数计算的所有本征值之积 = 矩阵对角线元素之积（上三角时）

```
det(T) = λ₁ × λ₂ × ... × λₙ
```

### 💡 小白理解

行列式告诉你这个变换把**体积放大（或缩小）了多少倍**。

---

## 10.3 重要性质

### 性质 1

```
算子可逆 ⟺ 行列式 ≠ 0
```

### 性质 2

```
det(AB) = det(A) × det(B)
```

### 性质 3

```
trace(ST) = trace(TS)
```

---

## 10.4 几何意义（最酷的部分！）

> **体积公式**：
>
> ```
> volume(T(Ω)) = |det T| × volume(Ω)
> ```

### 💡 大白话

一个区域经过 T 变换后，它的体积等于原来的体积乘以 `|det T|`。

### 🎯 例子

| det T | 体积变化 |
|-------|----------|
| 2 | 放大 2 倍 |
| 0.5 | 缩小一半 |
| 0 | 压扁成 0 |
| -1 | 镜像，体积不变 |

---

## 10.5 积分中的变量替换

> **换元公式**：
>
> ```
> ∫[T(Ω)] f(y)dy = ∫[Ω] f(T(x)) |det T'(x)| dx
> ```

### 💡 小白理解

微积分里换元时为什么要乘那个导数？

因为变换会改变"**小块的体积**"，而行列式正好度量了这种改变！

### 🎯 例子

| 坐标系 | 额外因子 | 原因 |
|--------|----------|------|
| 极坐标 | r | `r dr dθ` 里的 r 就是行列式 |
| 球坐标 | ρ² sin φ | 体积元素的行列式 |

---

## 📝 本章小结

| 概念 | 通俗理解 |
|------|----------|
| 迹 | 本征值之和 = 算总账 |
| 行列式 | 本征值之积 = 体积变化率 |
| det ≠ 0 | 可逆 |
| 几何意义 | 体积放大倍数 |
| 换元积分 | 行列式度量体积变化 |
