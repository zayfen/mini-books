# 第五章：本征值、本征向量、不变子空间——算子的"灵魂"

> 这是全书最重要的概念！

## 5.1 不变子空间

如果一个子空间里的向量被 T 作用后还留在这个子空间里，这个子空间就是**不变的**。

### 🎠 例子：旋转木马

你坐在木马上，无论怎么转，你都在木马上。

木马这个平面就是**不变的**。

---

## 5.2 本征值和本征向量

### 📐 定义

如果有**非零向量** v 满足：

```
T(v) = λv
```

那么 λ 叫**本征值**，v 叫对应的**本征向量**。

### 💡 大白话

有些特殊的向量被算子作用后**方向不变**，只是被**拉长了 λ 倍**。

（λ 可以是负数，表示方向反过来）

### 🎯 例子

| 变换 | 本征值 | 本征向量 |
|------|--------|----------|
| 把 x 方向拉长 2 倍 | 2 | 沿 x 轴的向量 |
| 旋转 90° | 无 | 没有向量方向不变 |

### 🔍 判断方法

```
本征值 = 使 T - λI 不可逆的 λ
```

---

## 5.3 复向量空间上的神奇定理

> **每个复向量空间上的算子都有本征值**

这是本书第一个高潮，证明完全不用行列式！

### 🌟 小白理解

在复数世界里，任何变换都至少有一条"**不动方向**"（虽然可能会被拉长）。

---

## 5.4 上三角矩阵

有了本征值，我们可以找到一组基，让算子的矩阵变成**上三角形式**。

### 📊 上三角的样子

```
⎡ λ₁  *   *   *  ⎤
⎢ 0   λ₂  *   *  ⎥
⎢ 0   0   λ₃  *  ⎥
⎣ 0   0   0   λ₄ ⎦
```

对角线下方全是 0，对角线上是本征值。

### 🧹 生活类比

就像把一堆乱糟糟的数据整理成表格，上半部分还有内容，下半部分全空了。

---

## 5.5 可对角化

如果存在一组由本征向量组成的基，那么算子关于这组基的矩阵就是**对角矩阵**。

### 📊 对角矩阵的样子

```
⎡ λ₁  0   0   0  ⎤
⎢ 0   λ₂  0   0  ⎥
⎢ 0   0   λ₃  0  ⎥
⎣ 0   0   0   λ₄ ⎦
```

只有对角线有数（都是本征值），其他位置全是 0。

### ✅ 可对角化的条件

算子可对角化 ⟺ 有足够多的本征向量

---

## 📝 本章小结

| 概念 | 通俗理解 |
|------|----------|
| 不变子空间 | 变换后还在里面的子空间 |
| 本征值 | 拉长的倍数 |
| 本征向量 | 方向不变的向量 |
| 上三角矩阵 | 整理过的"半空"矩阵 |
| 可对角化 | 能变成对角矩阵 |
