# 第三章：线性映射——把空间变来变去的魔术

## 3.1 什么是线性映射？

线性映射就是一个从向量空间 V 到向量空间 W 的函数 T，它满足两条规则：

### 📏 两条黄金规则

| 规则 | 公式 | 通俗解释 |
|------|------|----------|
| 加法保持 | T(u + v) = T(u) + T(v) | 整体扔进去 = 分别扔进去再相加 |
| 数乘保持 | T(λv) = λ T(v) | 先放大再变形 = 先变形再放大 |

### 🎪 生活类比

线性映射就像一台"**向量变形机**"。

你扔进去一个向量，它吐出来另一个向量。而且这台机器很"诚实"——

- 多个向量一起扔进去的结果，等于分别扔进去再相加
- 放大后再扔进去，等于扔进去后再放大

### 🎯 例子

| 映射 | 类型 |
|------|------|
| 旋转 | ✅ 线性 |
| 拉伸/压缩 | ✅ 线性 |
| 投影 | ✅ 线性 |
| 平移 | ❌ 非线性 |

---

## 3.2 零空间——被机器吞掉的向量

**零空间**就是所有被 T 变成 0 的向量的集合。

```
零空间 = {v | T(v) = 0}
```

### 📸 例子

把三维物体拍成二维照片的**投影**中：

- 所有垂直于屏幕的向量都会变成 0
- 这些向量就是零空间

### 🔍 零空间告诉我们什么？

零空间的大小告诉我们：**有多少信息被"吞掉"了**。

| 零空间 | 含义 |
|--------|------|
| 只有 {0} | 没有非零向量被吞掉（单射） |
| 很大 | 很多向量被映射成 0 |

---

## 3.3 值域——机器能吐出来的东西

**值域**就是所有可能的输出向量的集合。

```
值域 = {T(v) | v 在 V 里}
```

它告诉你这台机器**能生产出哪些产品**。

### 🎯 例子

- 3D → 2D 投影的值域是整个屏幕平面
- 旋转的值域是整个空间

---

## 3.4 线性映射基本定理（最重要的公式）

> **维数公式**：
> 
> ```
> dim V = dim(零空间) + dim(值域)
> ```

### 💡 大白话

> **输入空间的维数 = 被吞掉的维数 + 能吐出来的维数**

### 📊 图示

```
输入空间 (3维)
    │
    ▼
┌─────────────┐
│  线性映射 T  │
└─────────────┘
    │
    ├──► 零空间 (1维) ──► 被吞掉了
    │
    └──► 值域 (2维) ──► 能吐出来
```

---

## 3.5 单射与满射

### 单射（一对一）

不同的输入不会变成相同的输出（不会"撞车"）。

**判断方法**：单射 ⟺ 零空间只含 0

### 满射（满的）

能生产出所有可能的产品（值域等于整个 W）。

**判断方法**：满射 ⟺ 值域 = W

### 📊 总结表格

| 类型 | 条件 | 通俗理解 |
|------|------|----------|
| 单射 | 零空间 = {0} | 不会撞车 |
| 满射 | 值域 = W | 能生产所有产品 |
| 双射 | 单射 + 满射 | 完美对应 |

---

## 3.6 矩阵——线性映射的记账本

选好基之后，每个线性映射都可以写成一个矩形表格——**矩阵**。

### 📝 矩阵的列

矩阵的每一列就是**每个基向量被映射后的坐标**。

### 🔄 矩阵乘法

先做 T 再做 S 的复合映射，对应的矩阵就是 **S 的矩阵 × T 的矩阵**。

> 注意顺序：先作用的在右边！

---

## 3.7 可逆与同构

### 可逆映射

存在一个"反函数"能把变形后的向量变回去。

### 同构

两个空间之间存在可逆的线性映射 → 它们**本质上是同一个空间**，只是元素名字不同。

### 🔑 核心结论

> **两个有限维向量空间同构 ⟺ 它们的维数相同**
