# 第七章：内积空间上的算子——好算子的分类

## 7.1 伴随 T*

**伴随** T* 是满足以下条件的算子：

```
⟨Tv, w⟩ = ⟨v, T*w⟩  （对所有 v, w）
```

### 🪞 生活类比

T* 就像是 T 的"**镜像**"，把作用从右边移到左边。

---

## 7.2 自伴算子 T = T*

自伴算子就像**实数**（因为实数的共轭等于自身）。

### ✨ 特点

- 本征值都是**实数**
- 在复空间上，`⟨Tv, v⟩` 是实数 ⟺ T 自伴

### 🎯 例子

对称矩阵对应的算子就是自伴的。

---

## 7.3 正规算子 TT* = T*T

自伴算子是正规的**特例**。

### ✨ 好性质

- 不同本征值的本征向量互相**垂直**
- `‖Tv‖ = ‖T*v‖`

---

## 7.4 谱定理（全书高潮！）

### 复空间

> **正规算子 ⟺ 有规范正交本征向量基**

### 实空间

> **自伴算子 ⟺ 有规范正交本征向量基**

### 💡 大白话

"好算子"（正规/自伴）一定能找到一组**完美的基**，让它的矩阵是**对角线**。

### 🎯 意义

这是线性代数最漂亮的结果之一！

---

## 7.5 正算子

T 是正的 ⟺ 自伴且 `⟨Tv, v⟩ ≥ 0`。

### 🔢 类比

正算子就像**非负数**。

### ✨ 性质

每个正算子都有**唯一的正平方根**（就像每个非负数有唯一的非负平方根）。

---

## 7.6 等距同构

等距同构就是**保长度**的算子：

```
‖Sv‖ = ‖v‖  （对所有 v）
```

### ✅ 等价条件

```
S*S = I （恒等算子）
```

### 🎯 例子

- **旋转**：不改变向量的长度
- **反射**：不改变向量的长度

---

## 7.7 极分解

每个算子 T 都可以写成：

```
T = S × √(T*T)
```

- S 是等距同构（旋转/反射）
- √(T*T) 是正算子（拉伸）

### 🔢 类比

就像每个复数可以写成 `z = e^(iθ) × r`（旋转 × 拉伸）。

---

## 7.8 奇异值分解（SVD）

每个算子 T 都可以写成：

```
Tv = s₁⟨v, e₁⟩f₁ + s₂⟨v, e₂⟩f₂ + ... + sₙ⟨v, eₙ⟩fₙ
```

- s₁, ..., sₙ 是**奇异值**（正数）
- e₁, ..., eₙ 和 f₁, ..., fₙ 是两组**规范正交基**

### 💡 大白话

任何变换都可以分解成：

1. 先**投影**到 e 方向
2. **乘以**拉伸系数 s
3. 再**映射**到 f 方向

### 🎯 应用

奇异值分解在**数据压缩**、**推荐系统**、**图像处理**中都有广泛应用。

---

## 📝 本章小结

| 概念 | 通俗理解 |
|------|----------|
| 伴随 | T 的"镜像" |
| 自伴 | 像实数 |
| 正规 | 好算子 |
| 谱定理 | 好算子能对角化 |
| 正算子 | 像非负数 |
| 等距同构 | 保长度（旋转/反射） |
| 极分解 | 旋转 × 拉伸 |
| 奇异值分解 | 最重要的分解 |
