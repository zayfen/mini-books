# 前言：这本书有什么不同？

通常的线性代数书一开始就讲行列式，又复杂又难懂。但这本书反其道而行之——**行列式放到最后一章**！

前面的内容完全不需要行列式，直接从更直观的向量和映射开始。这样学起来更自然，也更容易理解线性代数的本质。

## 🗺️ 学习路线图

```
向量空间 → 基与维数 → 线性映射 → 本征值
    ↓
内积空间 → 谱定理 → 奇异值分解 → 行列式
```

## 🎯 我们的目标

不是让你背诵公式，而是让你真正**理解**线性代数的思维方式。
