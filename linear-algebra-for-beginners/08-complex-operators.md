# 第八章：复向量空间上的算子——深入内部结构

## 8.1 广义本征向量

有时候本征向量不够用，我们需要**广义本征向量**。

### 📐 定义

满足 `(T - λI)^j v = 0` 的向量。

### 💡 生活类比

- 本征向量是"**一次就死**"的
- 广义本征向量是"**慢点死**"的——多作用几次才变 0

---

## 8.2 广义本征空间分解

### 🌟 最重要定理

复空间上的每个算子，空间 V 可以分解为：

```
V = G(λ₁, T) ⊕ G(λ₂, T) ⊕ ... ⊕ G(λₘ, T)
```

每个 G(λ, T) 是相应于本征值 λ 的**广义本征空间**。

### 💡 大白话

整个空间可以**切成几块**：
- 每块对应一个本征值
- 每块里算子只和这个本征值有关

---

## 8.3 幂零算子

如果 N 的某次幂等于 0，N 就叫**幂零算子**。

```
N^j = 0 （某个 j）
```

### 🗑️ 生活类比

就像"**一次性用品**"，用几次就消失了。

---

## 8.4 凯莱-哈密顿定理

每个算子都满足它自己的**特征多项式**：

```
q(T) = 0
```

### 🔄 小白理解

算子和它的特征多项式之间有种神奇的"**自我指涉**"关系。

---

## 8.5 极小多项式

使 `p(T) = 0` 的**次数最小**的首一多项式，叫**极小多项式**。

### 📝 性质

- 极小多项式整除特征多项式
- 极小多项式的根 = 本征值

---

## 8.6 若尔当形

每个复空间上的算子都能找到一组基，让它的矩阵是**若尔当形**。

### 📊 若尔当块的样子

```
⎡ λ  1  0  0  ⎤
⎢ 0  λ  1  0  ⎥
⎢ 0  0  λ  1  ⎥
⎣ 0  0  0  λ  ⎦
```

- 分块对角矩阵
- 每块对角线相同（都是 λ）
- 对角线上方紧邻的线全是 1
- 其余都是 0

### 🧩 生活类比

就像把算子分解成一些**标准零件**，每个零件都长得差不多。
