# Mini Books 📚

通俗易懂的 mini 学习书籍系列，让每个人都能轻松掌握核心知识。

## 📖 书籍列表

### [线性代数应该这样学（小白版）](./linear-algebra-for-beginners/)

> 只需要初中数学知识，就能读懂的线性代数入门书

- **特点**：绕开晦涩的行列式，从直观的向量和映射开始
- **适合人群**：线性代数初学者、想重新理解线性代数的程序员
- **章节**：10章 + 前言 + 结语

### [AI Agent 开发应该这样学（小白版）](./agent-develop-for-beginners/)

> 从聊天窗口到真实世界的"数字打工猴"，让 Agent 编程像搭积木一样简单

- **特点**：去伪存真，专注 Python + LangGraph / OWL 前沿多模态智能体框架实战拆解
- **适合人群**：AI 开发者初学者、希望将自动化落地到业务的 Python 程序员
- **章节**：7章 + 前言 + 结语
